Front-end

1.This is a React and typescript based frontend. 
3.This frontend should be able to be hosted locally in port 3000.
4.This application should be able to generate marketing reports like product brochure, company brochure in pdf format.
5.This should have necessary modern npm packages for generating pdfs files. 
6.The way the pdf files are generated are as follows. 
6.1. The user will populate certain text fields in the front end. 
6.2. The user will also be able to upload certain images into the frontend.
6.3. There will be a button to generate the pdf file (e.g., product brochure) and it will compile the use inputs into a final pdf file.
